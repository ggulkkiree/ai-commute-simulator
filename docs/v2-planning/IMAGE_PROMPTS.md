# Image Prompts

## 범위 태그

- `[core]`: v2 MVP에 반드시 구현한다.
- `[optional]`: 나중에 확장 가능하며 핵심 제출 흐름에는 넣지 않는다.
- `[legacy/exclude]`: v2 핵심 구현 대상에서 제외한다.

공통 스타일 기준:

`Cute 2D casual webtoon style, warm educational game, bright colors, friendly characters`

v2에서는 기존 v1의 이미지 역할을 유지하되, 필요한 경우 아래 프롬프트로 일관된 새 이미지 세트를 생성할 수 있다.

## 자체 검증

- v1 assets 폴더에서 확인된 주요 이미지 역할을 모두 문서화했다.
- 화면별 필수 이미지, 활동 컷신, 아이템, 통근 장면, 결과 이미지를 `[core]`로 표시했다.
- 직무 수행 용도는 `[optional]`로 분리했다.
- 모든 프롬프트에 지정된 공통 스타일 문구를 포함했다.
