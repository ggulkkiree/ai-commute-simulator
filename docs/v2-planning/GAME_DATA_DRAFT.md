# Game Data Draft

이 문서는 v2에서 데이터 파일로 분리할 후보를 정리한 초안이다. 현재 v1 코드의 텍스트 일부는 인코딩 깨짐이 있으므로, v2에서는 아래 의미 단위로 UTF-8 한국어 문구를 정리해야 한다.

## 범위 태그

- `[core]`: v2 MVP에 반드시 구현한다.
- `[optional]`: 나중에 확장 가능하며 핵심 제출 흐름에는 넣지 않는다.
- `[legacy/exclude]`: v2 핵심 구현 대상에서 제외한다.

## [core] 확정 상태값

- `wokeUpEarly`: 아침 알람에서 바로 일어나면 `true`, 10분 더 자기 선택 시 `false`.
- `bagChecked`: 필수 준비물을 모두 챙긴 뒤 가방 완료 버튼을 클릭하면 `true`.
- `rawJudgment`: 저장/분석용 원본 결과. 값은 `success`, `incomplete`, `late`, `fail`.
- `displayJudgment`: 화면 표시용 결과. `rawJudgment = incomplete`이면 `displayJudgment = success`로 표시 가능.

## 화면별 텍스트

### [core] 학생 선택

- 제목: AI 출근 시뮬레이터
- 부제: 오늘도 출근 성공!
- 안내: 출근 연습할 학생을 선택해주세요. 최대 5명.
- 빈 상태: 등록된 학생이 없습니다. `[optional/teacher-settings]` 관리자 설정에서 학생을 추가해주세요.
- 버튼: 출근 시작하기, 교사용 설정, 내용 안내
- 확인 모달: `{학생명} 학생으로 시작할까요?`
- 확인 모달 버튼: 시작하기, 다시 선택

### [core] 출근 브리핑

- 제목: 오늘은 이렇게 출근해요!
- 핵심 문장: `{출근시간}까지 도착하려면 {출발시간}에 출발해요. {버스번호}번 버스를 타고 회사까지 가요.`
- 요약 항목: 출근 시간, 추천 출발 시간, 총 이동 시간, 타야 할 버스
- 경로: 우리 집 -> 출발 정류장 -> 버스 -> 도착 정류장 -> 본하이리
- 버튼: 출근 정보 입력하기

### [core] AI 출근 계획 입력

- 제목: AI에게 출근 정보를 알려주세요.
- 입력 항목: 출근 시간, 출발 정류장, 도착 정류장, 버스 번호, 버스 도착 시간, 버스 이동 시간
- 안내: 출근 정보를 모두 입력해주세요.
- 준비 완료 안내: 정보가 준비되었어요. AI가 계산해주기를 눌러주세요.
- 버튼: 출근 정보 다시 보기, AI가 계산해주기, 출근 정보 입력 완료

### [core] AI 출근 계획 결과

- 제목: AI 출근 계획이 완성되었어요.
- 요약: 기상 시간, 집에서 출발, 버스 탑승 시간, 회사 도착 예상
- 타임라인: 일어나기 -> 집에서 출발 -> 버스 탑승 -> 버스에서 내리기 -> 회사 도착
- 버튼: 입력 다시 보기, 전날 준비 시작하기

### [core] 전날 준비

- 버스 확인 제목: 내일 탈 버스 정보를 확인해요.
- 가방 제목: 내일 챙길 물건을 가방에 미리 넣어요.
- 알람 제목: 몇 시에 일어나야 할까요?
- 버튼: 확인, 다시 듣기, 가방 완료, 알람 맞추기

### [core] 수면 전환

- 알람 설정 완료: `{알람시간}에 알람이 울려요.`
- 잠들기: 캐릭터가 잠들었어요.
- 시간 흐름: 밤에서 새벽으로 시간이 지나가요.
- 아침: 곧 알람이 울려요.

### [core] 아침 알람

- 제목: 좋은 아침이에요.
- 현재 알람 시간 표시: `{알람시간}`
- 버튼: 바로 일어나기, 10분 더 자기
- 10분 더 자기 피드백: 10분 더 잤어요.
- 상태값: 10분 더 자기 선택 시 `wokeUpEarly = false`.

### [core] 아침 준비

- 메인 질문: 뭘 할까요?
- 안내: 원하는 활동을 선택해 아침 준비를 해요.
- 활동 버튼: 샤워하기, 아침 먹기, 양치하기, 옷 입기, 가방 챙기기
- 완료 안내: 아침 준비 완료! 출근을 시작해요.
- 정보 버튼: 오늘 출근 정보
- 도움말 버튼: 도움말
- 상태값: 필수 준비물 완료 + 가방 완료 버튼 클릭 시 `bagChecked = true`.

### [core] 출근 이동

- 제목: 출근길 시간 관리
- 대시보드: 현재 시간, 예상 도착, 목표 출근, 남은 시간
- 상태 문구:
  - 집에서 출발 준비
  - 정류장까지 걸어가요
  - 버스 번호를 확인해요
  - 버스에 탔어요
  - 내릴 정류장을 확인해요
  - 본하이리까지 걸어가요
  - 회사에 도착했어요
- 버튼: 걷기, 버스 타기, 기다리기, 하차벨 누르기, 내리기, 회사까지 걷기

### [core] 결과

- 결과 제목: 출근 결과
- 핵심 시간: 목표 출근 시간, 실제 도착 시간
- 섹션: 오늘의 결과, 내가 잘한 점, 다음에 해볼 점, 아쉬웠던 점, 선생님 한마디
- 버튼: 결과 다시 듣기, 자세한 행동 기록 보기, 다시 하기, 교사용 리포트 보기, 처음으로
- 결과 표시: `rawJudgment`와 `displayJudgment`를 분리한다.

### [core] 교사용 리포트

- 빠른 리포트 제목: 출근 시뮬레이터 리포트
- 상세 리포트 제목: 교사용 리포트
- 탭: 개별 보고서, 전체 요약
- 버튼: 이전 화면으로, 리포트 인쇄, 다시 하기, 학생 선택 화면으로 이동
